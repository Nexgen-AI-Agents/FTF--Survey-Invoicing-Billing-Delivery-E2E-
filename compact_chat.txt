HOW TO MAINTAIN THIS FILE
================================================================================

Whenever /compact is used:
  1. Note the current time (ET)
  2. Estimate tokens before: check if near context limit (~180,000-195,000)
     OR use previous token_usage.txt daily total as a proxy
  3. Estimate tokens after: typically 15,000-25,000 (summary size)
  4. Add a new COMPACT entry under today's DATE block
  5. Update DATE TOTAL and CUMULATIVE SUMMARY
  6. Run: git add compact_chat.txt && git commit -m "chore: update compact log [skip ci]" && git push origin main

Auto-update rule:
  This file is also updated by the Stop hook (scripts/token_log_hook.py)
  and the mandatory CLAUDE.md rule — no manual action needed.
